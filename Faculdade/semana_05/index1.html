Formulários

  • Atributos "action" e "method" do form:
    action - define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados
    method - define qual o método HTTP para enviar os dados (ele pode ser GET ou POST)

  • <\label> é vinculado a um campo do formulário, faz referência ao id do campo correspondente. Permite que o usuário clique no rótulo para ativar o campo correspondente.

  • <\input> entrada de texto de linha única. Seu atributo mais importante é o "type", o qual define a forma como o elemento input se comporta.
    Pode ser "text", "email", "password"
    Para definir o valor padrão de um elemento input, você tem que usar o atributo value.

  • <\textarea> input porém maior e é necessário tag de fechamento

  • <\button> o botão pode ser de três tipos: submit, reset ou button.
    submit envia os dados do formulário para a página de web definida pelo atributo action do elemento form
    reser redefine imediatamente todos os campos do formulário para o seu valor padrão
